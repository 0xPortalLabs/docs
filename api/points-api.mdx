## Endpoints

- [Get Points for Address](/api-reference/points/get-points-data-for-a-specific-address): Retrieve points data for a specific address across one or more protocols
- [List Available Adapters](/api-reference/points/list-all-available-adapters): Get a complete list of all supported protocol adapters

## Rate Limiting

The Points API implements a "soft" ratelimit of 50 requests per 10 seconds window. If you require higher limits, please reach out to us.
