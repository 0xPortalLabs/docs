## Endpoints

- [Get Deposit Claim](/api-reference/claims/get-deposit-claim): Verify a user's current points and
  get a signature for a deposit transaction
- [Get Settlement Claim](/api-reference/claims/get-settlement-claim): Verify that an offer was
  properly settled onchain and get a signature for a crosschain settlement transaction

## Rate Limiting

The Oracle API implements rate limiting to ensure fair access for everyone. If you require higher
limits, please reach out to us.
